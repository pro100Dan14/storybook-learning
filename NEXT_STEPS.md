---

### Шаг 5.2: Подготовка сервера

**У вас уже есть сервер?** (VPS, облачный сервер с Linux)

- **Да** → переходим к шагу 5.3
- **Нет** → нужно сначала купить/настроить сервер

**Требования к серверу:**
- Linux (Ubuntu, Debian, CentOS)
- SSH доступ
- Docker (рекомендуется) или Node.js
- Минимум 2GB RAM, 5GB места

---

### Шаг 5.3: Настройка SSH ключа для деплоя

Нужен отдельный SSH ключ, который GitHub Actions будет использовать для подключения к серверу.

---

## Что делаем дальше?
